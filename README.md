# 

## Type
为了保证所有的数据能够通过至多一次SRAM访问完成读写，仅支持32 bit类型，也就是4B

## variable
存放在一块固定的SRAM区域中


## 语法
1. 变量定义和赋值（所有变量需要预先定义然后使用）
```
var a,b;
var c;
a = 3*5;
var x = 8;
```
注意：你需要实现乘法，比如 3\*5需要优化成 "3\*1 + 3\*4", 然后翻译成riscv的时候用移位和相加实现

所有变量都是全局作用域
2. if语句

```
if a> 3{
    b = 5;
    if(b < 0){
        c = 4;
    }
    else{
        c = 3;
    }
}
```

3. while循环和for循环, 以及数组
```
var i;
var a[10];
for(i=0; i<10; i = i+1){
    a[i] = 8;              
}
```
注意：在RGparser.py中，当前定义一个数组后，会产生很多个变量，比如当前的例子中会产生 a@0, a@1, ... , a@9 变量，但是因为要支持的数组索引是固定值，需要使用 找到 a@0 的地址，然后加上一些偏移来访问第 i 个元素

4. 指针和解引用
```
var p;
var x = 3;
p = &x;
var y;
y = *p;
```

5. label和goto:

```
var i = 0;
while(i < 3){
    i = i+1;
    if i >=2 {
        goto out;
    }
}
out:
i = 7;
```
