# 

## Type
为了保证所有的数据能够通过至多一次SRAM访问完成读写，仅支持int类型

## variable
SRAM划分一块地址存储变量。不支持GC （甚至不支持scope），按照变量出现的先后顺序增加


## hardware
1. 分配 logic [40][30] color [15:0], 存储每一个位置的颜色像素
2. 外层game clock计时器，在游戏启动的时候，在每周期读取一个特殊的寄存器，然后开始计数。当计数完成后，将clock1寄存器的值修改成1
3. 内层监听寄存器，在游戏中，有 `70%` 的时间处于busy loop中，用来监听上层的按键。当检测到按下时，跳出循环，并记录下本周期的按键值
